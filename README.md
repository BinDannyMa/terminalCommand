# terminalCommand
## Editing a file
1. creat a file and open it for editing:
  > vi
  example:
  > vi text
  you creat a file named text, and you can add the suffix but not nesessary.
2. show the file:
  > cat
  example:
  > cat text
  you show the "text" file in the terminal window, if it already exists.If not, it show error.
